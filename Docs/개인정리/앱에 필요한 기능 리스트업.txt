하이브리드 앱에 별도로 들어가야되는 디자인 및 기능
	====== 코인 고스트에 필요하다고 생각되는 기능 ======
	- 네이티브 영역에 대한 아이폰과 안드로이드 디자인 가이드 필요
		* 아이폰은 Point단위
		* 안드로이드는 DP단위
		* 네이티브 영역
			* 로그인 / 회원가입
			* GNB 영역
			* 하단 푸터
			* 새창을 표현 할 수 있는 웹 화면
				* 하단의 네비게이션 (뒤로가기, 앞으로가기, 새로고침, 모바일 브라우저 앱으로 연결)
				* 더보기에서는 현재 보고 있는 페이지를 브라우저로 이동
			* 스와이프 메뉴
				* 단말기 상단 노치 디자인으로 인해서 Blur UI가 어색해 질 수 있음.
			* 검색화면
				* 검색어 자동 완성 
			* 마이페이지
			* 지갑
			* 푸시에 대한 알림 내역
			* 설정 (마이페이지에 포함되어도 됨.)
				- 마이페이지 화면에 들어가거나 별도 탭으로
				- 앱에 필요 한 설정
					* 알림설정
					* 마케팅 푸시 이용 동의
					* 이용가이드
						* 이미지를 활용해서 페이징 형태로 기능 제공
					* 자주 묻는 질문
					* 1:1문의/건의
					* 고객센터
					* 이용약관
					* 개인정보이용 
					* 버전 정보 (구버전을 사용하고 있다면 업데이트 버튼을 노출)
			* 디자인 팝업
			* 오류 표시용 토스트
			* 업데이트 팝업
			* 권한 팝업 (국내 서비스)
				* 앱에서 사용 하는 권한에 대한 내용 고지
			* 로딩에 대한 프로그래스 표현
				* 따로 디자인이 된다면 프레임 애니메이션 형태로 만들 수 있습니다
		* 플랫폼에 맞게 기능 분리
			* 앱 종료
				* 안드로이드 : 소프트웨어 키를 활용해서 종료 기능 추가 가능
				* 아이폰 : 앱 종료 기능 불가능
		* 업데이트 기능 (버전 체크)
			* 강제 업데이트
			* 선택 업데이트

	=== 네이티브 앱을 활용 했을 때 진행 할 수 있는 기능
	- 푸시
		* 이미지를 함께 전송하는 푸시도 개발 가능
	- 앱 스키마
	 	* 푸시를 활용해서 특정 화면을 바로 진입
	 	* 모바일 웹에서 앱으로의 이동
	 		예를들면 코인고스트 지갑기능을 앱으로만 사용 할 수 있다고 할때
	 		모바일 웹에서는 지갑에 접근 할 때 앱을 실행
